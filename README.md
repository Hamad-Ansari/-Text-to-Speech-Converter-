# 🧠 AI Text-to-Speech Studio

🚀 A modern, cloud-ready **Text-to-Speech (TTS) SaaS application** built with Streamlit and OpenAI.
Convert text or PDFs into natural-sounding speech with multiple voices and languages.

---

## ✨ Features

* 🔊 **Text to Speech** — Convert any text into high-quality audio
* 📄 **PDF to Speech** — Upload PDFs and convert them into speech
* 🌍 **Multi-language Detection** — Automatically detects input language
* 🎙️ **Multiple Voice Styles** — Choose from different AI voices
* 🎧 **Audio Playback + Download** — Listen and download instantly
* ☁️ **Cloud Compatible** — Works seamlessly on Streamlit Cloud

## 📦 Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Add Requirements File

Create `requirements.txt`:

```txt
streamlit
openai
pypdf
langdetect
```

---

### 4️⃣ Run the App

```bash
streamlit run app.py
```

---

## 🔑 Usage

1. Enter your **OpenAI API Key**
2. Choose:

   * Model (tts-1 / tts-1-hd)
   * Voice style
   * Output format
3. Input:

   * ✍️ Text OR
   * 📄 Upload PDF
4. Click **Generate Speech**
5. 🎧 Listen or ⬇️ Download
